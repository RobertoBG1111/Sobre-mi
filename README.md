# Sobre-mi
👋 Hola, soy Roberto.

Ingeniero automotriz en transición hacia el análisis de datos. Mi formación técnica y mi experiencia en proyectos como Formula SAE y Baja SAE me dieron las bases para abordar problemas complejos con método; mi paso por el emprendimiento me enseñó a tomar decisiones con información incompleta. Hoy combino ambas perspectivas en el análisis de datos.

Lo que hago:

- Análisis exploratorio y estadístico con Python (pandas, scipy, matplotlib)
- Consultas y modelado de datos en SQL
- Visualización y dashboards en Tableau
- Aplicación de metodologías Lean Six Sigma (Green Belt) a problemas analíticos
- Uso de herramientas de IA (Claude, Gemini, Perplexity) como apoyo para profundizar cada análisis: validar hipótesis, explorar enfoques estadísticos alternativos y llevar cada proceso un nivel más allá del análisis superficial

En qué estoy trabajando:

Bootcamp de Análisis de Datos en TripleTen, desarrollando proyectos que aplican el análisis de datos a problemas reales, desde movilidad urbana y productividad económica en Latinoamérica hasta optimización de procesos.
Busco oportunidades como Data Analyst Jr. donde pueda aportar pensamiento estructurado, honestidad analítica y ganas de crecer.
